# ⚡ Productivity Prompts Pack

## 1. Weekly Planning System

```
Create a comprehensive weekly plan for [目標/項目]。

Include:
1. Weekly Goals (3-5 SMART goals)
2. Daily Action Items (Mon-Fri)
3. Time Blocking Schedule
4. Buffer Time Allocation
5. Weekly Review Questions

Format as a structured template I can copy.
```

## 2. Meeting Agenda Builder

```
Generate a meeting agenda for [會議類型]。

Duration: [X] minutes
Participants: [參加者]
Goal: [會議目標]

Please create:
1. Opening (5 min) - Objectives
2. Main Topics (X min each)
3. Discussion Questions
4. Action Items
5. Closing - Next Steps

Add facilitation tips for each section.
```

## 3. Inbox Zero System

```
Create an email management system using [工具：ChatGPT/Claude]。

Requirements:
- Process 50+ emails/day
- Prioritize by urgency
- Auto-generate responses
- Create task follow-ups

Provide:
- Morning routine (30 min)
- Afternoon routine (15 min)
- Response templates
- Automation rules
```

## 4. Content Calendar Creator

```
Create a 30-day content calendar for [平台/主題]。

Platform: [的平台]
Niche: [乜嘢領域]
Posting frequency: [每週幾多篇]

For each week provide:
- Content themes
- Post ideas (5 each)
- Hashtag sets
- Engagement strategies
- Best posting times
```

## 5. Habit Tracker Framework

```
Design a habit tracking system for [想培養的習慣]。

Include:
1. Habit definition
2. Trigger cues
3. Reward system
4. Progress visualization
5. Accountability measures
6. Troubleshooting guide

Make it actionable and measurable.
```

## 6. Decision Matrix

```
Create a decision framework for [要做的決定]。

Options: [選項A, B, C]

Evaluate each on:
```
